What is FIX.YTM?

FIX.YTM is a browser extension that allows you to access extended YouTube Music functionality concerning your user-content interactions and some visual features. With it, you will finally be able to view content statistics, topic, genre, read and leave your own comments on it, view date of publication, and more. It also provides some visual utilities, such as sorting playlists to view most popular, most liked or recent songs, etc.

Solutions behind FIX.YTM

FIX.YTM met a bunch of challenges throughout its development and had to implement these solutions:

  • Authorization using OAuth2 & redirect flow
  • Accessing Google APIs using query HTTP requests
  • Filtering region restricted content utilizing HTTP requests to ipinfo.io
  • Using React and React DOM to render responsive overlay menus
  • OAuth2
  • Return authorization flow
  • Content filtering
  • Context-aware interface
  • Non-intrusive design
  • Aggressive application of React
