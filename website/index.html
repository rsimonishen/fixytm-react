FIX.YTM

Enhance your YouTube Music experience by adding missing features from the main YouTube platform

What is FIX.YTM?

FIX.YTM is a browser extension that allows you to access extended YouTube Music functionality concerning your user-content interactions and some visual features. With it, you will finally be able to view content statistics, topic, genre, read and leave your own comments on it, view date of publication, and more. It also provides some visual utilities, such as sorting playlists to view most popular, most liked or recent songs, etc.

Solutions behind FIX.YTM

FIX.YTM met a bunch of challenges throughout its development and had to implement these solutions:

  • Authorization using OAuth2 & redirect flow
  • Accessing Google APIs using query HTTP requests
  • Filtering region restricted content utilizing HTTP requests to ipinfo.io
  • Using React and React DOM to render responsive overlay menus
  • OAuth2
  • Return authorization flow
  • Content filtering
  • Context-aware interface
  • Non-intrusive design
  • Aggressive application of React

Authorized performance

You can perform authorised actions using FIX.YTM like placing your own comments, liking or disliking content (in future - even posting your own content) etc. Such actions require authorization via OAuth2. Browser extension requires authorization to perform authorised actions upon each reload of the page. The future desktop application, however, will be able to handle reauthorization automatically.

Privacy policy

Privacy policy of FIX.YTM browser extension dictates complete privacy. The browser extension may collect usage data, such as visits to its website or caching your requests, but the data utilization is kept strictly for the operation of the extension. The extension does not store, collect or sell your sensitive personal data to any third parties or use it for its own purposes. Read more about privacy policy of FIX.YTM
